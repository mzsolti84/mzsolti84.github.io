    What I did

    As a software developer and tester, I work on the following projects:

    Sudoku

    Sudoku API. GameBoard and solution generation using the "back-track" method.

    Simple Task Manager

    A simple task manager application made in Flutter framework. Implemented with BloC state management.

    ParaBank Test

    Automatic testing of https://parabank.parasoft.com webapp in Cypress framework. The test cases use the Given-When-Then (Cucumber) methodology and were written in TypeScript.

    Carnivora

    To present a data structure model with descendant dependencies, using the example of a cladistic classification method for predators. The management and understanding of the data structure is illustrated with an interactive genogram (cladistic genetic family tree) and additional administrative interfaces and functions.
